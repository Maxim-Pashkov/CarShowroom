# Car showroom Android app

[![Build status](https://build.appcenter.ms/v0.1/apps/3eb2eee6-9603-43f9-9964-228844ad3b60/branches/master/badge)](https://appcenter.ms)

To clone need write this in command line:
```
git clone https://github.com/Maxim-Pashkov/CarShowroom.git
```

## Dependencies

* Xamarin.Forms >= 3.4.0
* Xamarin.UITest >= 2.2.4
* SQLite
* NUnit 2.6.4

## We use:
* [Xamarin](https://dotnet.microsoft.com/apps/xamarin): for create Android app
* [NUnit](https://nunit.org/): for create Unit and UI tests
* [Xamarin.UITest](https://docs.microsoft.com/en-us/appcenter/test-cloud/uitest/): for create UI tests
* [App Center](https://appcenter.ms/): CI
